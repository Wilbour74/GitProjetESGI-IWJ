# Contributing Guidelines

This is a student project created for learning Git and GitHub basics.

## How to Contribute

1. Fork this repository
2. Create a new branch: `git checkout -b my-change`
3. Make your changes
4. Commit your work: `git commit -m "Describe your change"`
5. Push to your fork: `git push origin my-change`
6. Open a Pull Request

You can practice adding files, editing content, or fixing typos.

Thanks for learning with us!
